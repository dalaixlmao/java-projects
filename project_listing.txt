====================================
HOW TO USE
====================================

For any project:

1. Navigate to project directory:
   cd /home/user/java-projects/<category>/<project-name>

2. Review the Application.java file for TODO comments:
   cat src/main/java/com/<package>/Application.java

3. Build the project:
   mvn clean install

4. Run the application:
   mvn spring-boot:run

5. Implement the TODO sections according to requirements

====================================
EXAMPLE: Running dependency-analyzer
====================================

cd /home/user/java-projects/build-tools-assignments/dependency-analyzer
cat src/main/java/com/buildtools/dependencyanalyzer/Application.java
mvn clean install
mvn spring-boot:run
